# 1c_project
🌿 Mental Health Support Platform

High-Performance C++ Backend | Type-Safe TypeScript Frontend

Профессиональное решение для оказания психологической помощи студентам. Проект объединяет высокую производительность компилируемого бэкенда на C++ и надежность типизированного фронтенда на TypeScript для создания безопасного и отзывчивого сервиса.
🎯 Проблема и решение

В условиях высокой академической нагрузки студенты часто нуждаются в быстрой и анонимной поддержке. Наша платформа решает эту задачу через:

    Privacy-First: Никакого хранения логов в БД. Все данные сессии существуют только в RAM.

    Low Latency: Использование C++ минимизирует накладные расходы сервера при обработке запросов.

    Smart Prompting: Автоматическая адаптация контекста ИИ под эмоциональное состояние пользователя.

🎯 Проблема и решение

    Smart Prompting: Автоматическая адаптация контекста ИИ под эмоциональное состояние пользователя.

🔄 Схема прохождения данных

Ниже представлена логика движения информации от ввода пользователя до получения ответа от нейросети:

    Client (TS): Пользователь выбирает состояние (напр. "Stress") и вводит сообщение. Фронтенд валидирует типы и отправляет JSON-пакет.

    Server (C++): Бэкенд на базе Crow принимает запрос, парсит JSON и оборачивает данные в системный промпт (инструкцию для "психолога").

    AI Integration: Сервер выполняет асинхронный POST-запрос к API нейросети (OpenAI/Llama).

    Response Cycle: Полученный текст очищается от метаданных и возвращается клиенту для мгновенного отображения.

🏗 Архитектура

Система построена на принципах модульности и строгой типизации:

    Core Engine (Backend): Написан на C++20. Отвечает за маршрутизацию, управление HTTP-сессиями и сетевое взаимодействие с AI-провайдерами.

    User Interface (Frontend): Построен на TypeScript. Обеспечивает корректность передаваемых данных и динамическое обновление UI без перезагрузки страницы.

    Security Layer: Реализует концепцию In-memory storage, гарантируя, что личные сообщения не сохраняются на диске.

🛠 Технологический стек
Слой	Технологии
Backend	C++20, Crow Framework, CPR, nlohmann/json
Frontend	TypeScript, Vite, CSS Modules
Infrastructure	CMake, vcpkg, Docker
AI Layer	REST API (OpenAI-compatible)
🚀 Дорожная карта (Roadmap)

    [x] MVP: Базовый чат и интеграция с LLM API.

    [x] State Awareness: Система контекстных тегов для точного промптинга.

    [ ] Voice Interaction: Поддержка голосового ввода сообщений.

    [ ] Multi-Model Support: Возможность переключения между разными ИИ-движками на лету.

📦 Развертывание
Требования

    Компилятор с поддержкой C++20 (GCC 11+ / Clang 13+).

    Node.js v18.0+.

    Настроенный CMake и менеджер зависимостей vcpkg.

Установка

Проект состоит из двух частей: компиляции исполняемого файла сервера и сборки фронтенд-бандла. Подробные спецификации конфигурации доступны в технической документации внутри репозитория.

Данная платформа является инструментом поддержки и самопомощи. В критических ситуациях всегда рекомендуется обращение к сертифицированным специалистам.
